Guide

Base url: https://connect-hackaway.herokuapp.com/

Available endpoints

- All endpoints must be authenticated except /register and /login
  • /user - register user: POST
    • fields
    • firstName: String
    • lastName: String
    • email: String
    • password: String
  • /user/login - login user: POST
    • fields
    • email: String
    • password: String
  • /auth/{code} - authenticate user: POST. {code} from mono connect
    • For /auth/{code} headers: { Accept: "application/json", "Content-Type": "application/json", 'mono-sec-key': mono-app secret, ...default headers, },
